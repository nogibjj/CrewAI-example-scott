# CrewAI-example

## get start
### install the requirements packages
```
pip install -r requirements.txt 
```

### install Ollama
```
curl -fsSL https://ollama.com/install.sh | sh
```

#### install solar model from Ollama 
```
ollama pull solar
```
#### install openhermes model from Ollama
```
ollama pull openhermes
```


### run the file
```
python crewAI.py
```
